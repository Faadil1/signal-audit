SignalAudit

The Trading Gate

Before a strategy runs, the market must pass the audit.

Most trading tools ask what to buy.
SignalAudit asks whether the gate should open at all.

View source on GitHub
⌁
Checkpoint 01

Strategy arrives

A trader, or an autonomous agent, presents a strategy hypothesis at the gate. No execution has happened yet. No position is open.

◎
Checkpoint 02

Signals are audited

Five market signal categories are scanned — volume, derivatives, on-chain flow, sentiment, narrative velocity. Each one is checked against an explicit, named threshold.

⛊
Checkpoint 03

Gate opens or closes

The verdict is deterministic, not predictive. If the gate opens, the original strategy proceeds. If it closes, a modified, re-evaluable spec is issued instead.
